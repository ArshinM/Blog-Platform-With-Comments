📌 Blog Platform with Comments


Overview

This is a simple full-stack blogging platform that allows users to create posts and interact through comments. Users can register, log in, and manage their blog posts easily with a clean interface.


🚀 Features

User registration, login, and authentication
Create, edit, and delete blog posts
Comment section for each post
Display all blog posts in a feed
RESTful APIs for backend operations
Database integration for users, posts, and comments


🔄 Workflow

Login → Blog Dashboard → Create Post → View Posts → Add Comments → Interaction



🛠️ Tech Stack

Frontend: HTML, CSS, JavaScript
Backend: Node.js, Express.js
Database: MySQL / MongoDB


▶️ How to Run

Clone the repository
Install dependencies using:
npm install
Configure database connection in db.js
Start backend server:
node server.js
Open index.html in browser
